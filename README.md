# YMGMS

## Información del proyecto

Proyecto construido con Vite, React, TypeScript y shadcn-ui.

## ¿Cómo editar este código?

Puedes trabajar de varias formas:

**Usar tu IDE preferido**

Clona el repositorio y trabaja localmente. Solo necesitas tener Node.js y npm instalados - [instalar con nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Sigue estos pasos:

```sh
# Paso 1: Clona el repositorio.
git clone <TU_URL_GIT>

# Paso 2: Entra en el directorio del proyecto.
cd <NOMBRE_DEL_PROYECTO>

# Paso 3: Instala las dependencias.
npm i

# Paso 4: Inicia el servidor de desarrollo.
npm run dev
```

**Editar directamente en GitHub**

- Navega al archivo que quieras modificar.
- Haz clic en el botón "Edit" (icono de lápiz) en la parte superior derecha.
- Realiza los cambios y haz commit.

**Usar GitHub Codespaces**

- Ve a la página principal del repositorio.
- Haz clic en el botón "Code" (botón verde) cerca de la parte superior derecha.
- Selecciona la pestaña "Codespaces".
- Haz clic en "New codespace" para abrir un Codespace.
- Edita los archivos y haz commit y push cuando termines.

## Tecnologías del proyecto

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Despliegue

Para desplegar el proyecto, ejecuta `npm run build` y sube la carpeta `dist` generada a tu servicio de hosting preferido.
